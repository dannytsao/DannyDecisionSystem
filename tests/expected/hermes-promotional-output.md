## 【對我有用的重點】

### AI / 自動化

- 在下 prompt 前先寫清楚驗收標準，不讓 agent 自行猜測需求。
- 每次只交辦一個範圍明確的小任務。
- 部署前人工審查所有變更。

1. 建立包含目標、驗收標準與邊界條件的 agent 任務範本。
2. 將大型需求拆成輸入與輸出明確的小任務。
3. 在部署流程加入人工審查關卡。
